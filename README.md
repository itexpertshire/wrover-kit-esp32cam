# wrover-kit-esp32cam
Board : ESP32-WROVER-DEV 
Camera: OV2640 
OLED Display:

Features Implemented:
   1. Stream frames as fast as possible
   2. Display log message in Oled Display
   3. Control esp32 camera remotely using http call
